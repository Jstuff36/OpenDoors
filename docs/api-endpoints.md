# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `DELETE /api/users/:id`
- `PATCH /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

## Listings

### Listings reference the user table

- `GET /api/listings`
- `GET /api/listings/:id`

### References

- `GET /api/references`
- `POST /api/references`
- `PATCH /api/references/:id`
- `DELETE /api/references/:id`
